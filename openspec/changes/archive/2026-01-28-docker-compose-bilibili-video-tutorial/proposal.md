# Change: 在 Docker Compose 文档中添加 B站视频教程

## Why

当前 Docker Compose 文档仅包含文字和静态图片,缺少视频演示资源。添加 B站视频教程可以:
- 为视频学习者提供更直观的学习资源
- 通过动态演示降低 Docker Compose 的学习门槛
- 提升文档的实用性和用户满意度

## What Changes

在 `docs/installation/docker-compose.md` 文档中添加 B站视频演示内容:
- 在文档顶部(标题后)添加 B站视频嵌入组件
- 使用 Docusaurus 的 React 组件实现视频嵌入
- 添加适当的描述文字,说明视频内容和时长
- 确保视频展示不影响文档阅读体验
- 验证视频在本地和生产环境的显示效果

**技术要点**:
- 使用 iframe 嵌入 B站视频或链接卡片方式
- 保持响应式布局,适配不同设备
- 遵循项目文档风格和 frontmatter 规范
- 确保 `npm run build` 构建通过

## Impact

- **影响的规范**: 无(文档更新,不影响功能规范)
- **影响的代码**: `docs/installation/docker-compose.md`
- **测试要求**:
  - 本地运行 `npm start` 验证视频展示效果
  - 执行 `npm run build` 确保无构建错误
  - 检查视频在不同设备上的响应式显示
  - 验证外部链接的可访问性

## 风险与注意事项

- **外部依赖**: B站视频链接有效性需要持续维护
- **网络访问**: 部分地区可能无法访问 B站,需考虑添加备选方案说明
- **文档风格**: 视频嵌入需保持与现有文档风格一致
